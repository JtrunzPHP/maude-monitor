How to Read V3.4b

⭐ CAR Case Studies

Cumulative Abnormal Returns: stock return minus S&P 500 return around MAUDE signal dates. Academic event-study methodology.

PRR Signal Detection

FDA's Proportional Reporting Ratio. Detects disproportionate failure modes. PRR≥2 + χ²≥4 = FDA-grade signal.

Recall Cascade

Tracks MAUDE spike → Warning Letter → Recall progression. Late cascade = high probability of further action.

Severity-Weighted

Deaths×100 + Injuries×10 + Malfunctions×1. Captures impact, not just volume.

Cross-Company Signal

Ranks companies by composite of rate/10K + trend + z-score. Worst = short candidate.

Event-Date Smoothing

Strips batch-dump noise using date_of_event. "Reports" = raw. "Smoothed" = adjusted.

Multi-Signal Correlation

6 MAUDE signals tested against stock returns at 0-6mo lags. Confidence 0-100.

R-Score (0-100)

Composite risk. Z(20)+Trend(20)+Deaths(20)+Injuries(20)+Rate(20). R≥50=investigate, R≥70=act.
